"WP Post Status" allows posting a status message on the WordPress admin dashboard page. 

== Description ==

Adding status on the admin dashboard is made easy with this plugin. This plugin is for showing statu on the 
dashboard page. Administrators are allowed to add/edit the status and other options. Super Admin/Netwrok Admin
has capability to overwrite/update all subsite status. This plugin is intended to use on both singlesite and multisite.

Some features of this plugins are listed below:

* Add status and show it on dashboard page.
* Network admin has capability to overwrite/update all subsite status.

== Frequently Asked Questions ==

= Can this plugin be used on multisite? =

Yes, this plugin is intended to use on both singlesite and multisite.
